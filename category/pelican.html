用pelican搭建博客的一些问题及解决方法

Date 周五 11 七月 2014 By RONG Category Pelican. Tags Pelican Python

一直很想拥有自己搭建的网站，从网上了解到可以用python的框架搭建博客，正巧前段时间学了一下python，于是马上搜索各种网页框架，后来选用了web.py框架，并在网上看了一下Reddit创始人的博客建站视频，了解了下网页基础知识、实现了简单的数据库管理，并在GAE上搭建成功，但登陆GAE需要翻墙，后来为了方便便想迁移到国内，也考虑过SAE和BAE，但由于国内还有域名认证什么的，觉得麻烦便令求它法，后来知道了Pelican这个静态网页生成程序，同时又有Github这个平台可以搭建我的博客，因此决定用Pelican来搭建博客，至于如何使用Pelican和Github搭建博客，网上已经有了许大神们精辟的讲解，现在主要说下在windows下安装和使用Pelican可能会遇到的问题。

  • 安装Fabric时出现Unable to find vcvarsall.bat错误

当时试过了在网上找到的很多种方法，可悲的是都不能解决这个问题，经过苦苦的搜索，终于找到了解决的方法：下载安装VS2010后，在python安装目录下找到msvc9compiler.py这个文件，将其中

SET VS90COMNTOOLS=%VS008COMNTOOLS%

这条语句改为

SET VS90COMNTOOLS=%VS100COMNTOOLS%，然后就可以正常安装Fabric。

  • 删除Pelican生成页面原始的一些用来做example的文章

只需要在Pelican的pelicanconf.py中添加

PATH='你的博文等要处理的文件所在的文件夹'

这条语句是限定Pelican在生成页面时要处理的文件夹。
